### HUD 디자인

```
┌──────────────────────────────────────────┐
│  🔔 벨튀 12  │  🔥 콤보 x5  │  ❤️ 3  │  🏢 3F  │  ⏱ 01:34  │
```

- 하트 잃을 때 깨지는 애니메이션

### 시청자 참여 UI (채팅 연동)

화면 좌측에 **시청자 액션 피드**:
```
┌─────────────────┐
│ 💬 시청자 액션   │
│ ───────────────  │
│ user1: !함정 🍌  │
│ user2: !문열어 🚪│
│ user3: !경보 🚨  │
│ user4: !응원 ⚡  │
└─────────────────┘
```

문 위에 시청자 닉네임 표시:
```
   [user1님의 집]
  ┌──────────┐
  │          │ ◉ ← 벨
  │   🚪     │
  │          │
  └──────────┘
```

## 🅱 VRChat 월드 (스트리머 대전 버전) — 디자인 컨셉

![VRChat 버전 컨셉](/Users/mac/.gemini/antigravity/scratch/belltui-game/belltui_vr_concept_1777364692597.png)

### 아트 스타일: **한국 아파트 × 빌런 그래피티**

| 항목 | 방향 |
|:---|:---|
| **화풍** | 로우폴리 스타일라이즈드 3D. 카툰 렌더링 + 네온 포인트 |
| **시점** | 1인칭 VR (풀트래커 지원). 직접 뛰어다니는 체감 |
| **색감** | 회색 콘크리트 베이스 + 보라/빨강 네온 낙서. 형광등의 차가운 빛 |
| **분위기** | 밤에 홀로 아파트 복도를 걷는 긴장감 + 코믹한 주민 리액션 반전 |

### 환경 디테일

**복도 소품 (한국 아파트 감성)**:
- 현관문 옆 우유 배달함
- 택배 박스 더미 (숨는 곳으로 사용 가능)
- 소화기, 비상등
- 신발장 밖에 놓인 슬리퍼들
- 가스/수도 계량기
- 복도 끝 비상구 표시등 (초록)

**빌런 오염 요소 (쵸로키 테마)**:
- 벽에 스프레이로 그린 해머 마크
- "VILLAIN" / "빌런 조합" 스티커
- 네온 LED 테이프가 복도 모서리를 따라 붙어 있음
- 쵸로키 포스터 ("최강 빌런 모집 — 조건: 없음")

### VRChat 인터랙션

| 동작 | VR 입력 |
|:---|:---|
| 벨 누르기 | 손으로 직접 벨 버튼 터치 |
| 대시 | 양 팔 스윙 (또는 조이스틱) |
| 숨기 | 택배 박스 뒤로 웅크리기 (풀트래커 감지) |
| 해머 사용 | 등 뒤에서 해머를 꺼내 휘두르기 |
