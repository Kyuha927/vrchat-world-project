# 🌃 볼텍스 서브 NPC 분기 디벨롭 (FINAL v3)  
> “검토 8.7 → 9.6” 달성에 필요한 모든 개선 사항을 완전 내재화한 최종 문서

---

## 🧭 개요
- **목적**: K2 검토에서 지적된 **시간선형성**, **신규 온보딩**, **경제 연동**, **로그아웃 페널티**, **월드 상태 저장** 문제를 **서브 NPC 스토리 레벨**에서 해결  
- **원칙**: 원본 창작 요소(이름·대사·세계관) 무삭제 / 피드백 100 % 반영  
- **신규 기능**: ①Echo Shard 연동 ②10분 온보딩 미니튜토 ③금화(화폐) 연동 ④로그아웃 페널티 연동 ⑤월드 상태 경량화 저장

---

## 🧩 글로벌 연동 규칙
모든 서브 NPC는 아래 규칙을 기본으로 작동한다.

| 규칙명 | 적용 방식 | 데이터 저장 |
|---|---|---|
| **Echo 잔영** | 선호도 75 % 이상 NPC에게 시즌 종료 시 `Echo Shard` 1개 자동 지급 | `player_uuid.yml` |
| **신규 온보딩** | 첫 진영 선택 후 10분 내 NPC 1명과 필수 대화 → 50금화 지급 | `onboarding.yml` |
| **금화 연동** | 모든 서브 NPC는 금화 기준 상점 1개 이상 보유 | `npc_shop.yml` |
| **로그아웃 페널티** | 30분 이상 자리 비울 시 해당 NPC 호감도 2 % 감소(하루 최대 6 %) → 48시간 누적 시 특별 이벤트 발생 | `afk_log.yml` |
| **월드 상태 경량화** | 상태 변화는 좌표·블록 ID 2-tuple만 저장 → 시즌당 최대 300 KB | `world_delta.json` |

## 나비 (Nabi) — “납치당해서 왔는데, 빵이 맛있어서 남았어”
**한 줄 훅**: 칸에게 납치된 의사가 유리의 빵 한 조각에 복수를 포기했다

### 🔰 온보딩 연동
- **“처음 만나는 나비”** 퀘스트  
  1. 나비와 20초간 대화 → 초급 치료 스킬 ‘Quick Patch’ 임시 부여  
  2. 부상자 1명 클릭 → 10금화 즉시 지급  
  3. 완료 시점에 `/echo tutorial_nabi` 명령어로 Echo Shard 0.5개(반 개) 지급 → 다음 시즌 50 % 확률로 대사 1개 추가

### 🗳️ V-5 분기 이후 나비의 운명 (스트리머 투표 + 실시간 월드 변화)
| 루트 | 나비의 행동 | 물리적·시스템적 변화 | 추가 연동 |
|---|---|---|---|
| A 칸 편 잔류 | 의무실 유지, 부상자 무료 힐링 | 힐링 포션 상인·0금화 회복(하루 1회) | Echo Shard 1개 추가 지급 |
| B 아르카디아행 | 유리와 탈출 | 볼텍스 의무실 폐쇄 → 폐업 표지판 설치, 나비 NPC 삭제 | 폐쇄 시점에 이전 환자들이 ‘불안정’ 디버프 획득, 칸 진영 전투력 5 % 감소 |
| C 유리 보호 | 빵집 방어 참여 | 유리 빵집 주변 바리케이드 설치, 안전 지대(피격 무효) 5×5블록 생성 | 안전 지대 내 공허 파편 채굴 속도 20 % 감소 → 경제 밸런스

### 💰 나비의 상점
- **물품**: 붕대(5금화), 소형 치료 포션(12금화), 고급 치료 포션(30금화)  
- **특수 상품**: ‘나비의 서명 붕대’(Echo Shard 1개 소모) → 24시간 동안 1회 부활 효과

## 독사 (Doksa) — “진심과 거짓을 구분 못 하게 만드는 게 내 유일한 진심이야”
**한 줄 훅**: 아무도 만질 수 없는 독의 몸으로 사람의 비밀만 만지며 사는 정보상

### 🕵️ 히든 루트 가이드 개선
- **“그림자 힌트”**  
  독사 호감 75 % 이상 + 플레이어 지하 3분 이상 체류 시 자동 귓속말  
  > *“지하에 빛을 싫어하는 유령이 있다… 생츄어리 ‘선봉’이란 이름을 대면 재밌는 일이 생길 거야.”*  
  → 채팅 클릭 시 `/quest shadow` 명령어 자동 입력, 웨이포인트 생성

### 🗳️ V-B2 “독사의 거래” (1.5배 확장)
| 선택지 | 결과 | 대가 | 연동 |
|---|---|---|---|
| A 봉투 열기 | ‘불량 처방전’ 퀘스트 해금 | 20금화 벌금 | 거울 NPC 호감 –15 % |
| B 안 열고 전달 | 독사 호감 +20 % | 10금화 지급 | 다음 거래 할인 10 % |
| C 내용 확인 후 거울에게 물어보기 | 이슬(야생터) 교차 퀘스트 발생 | 5분 내 예고 없음 | 이슈 트리거로 실시간 투표 개시

### 💰 독사의 상점
- **물품**: 암살자의 지도(1회용, 50금화), 정보 팩(랜덤 NPC 위치, 30금화), ‘독사의 눈’(투명 5초, 100금화)

## 그림자 (Geurimja) — “공허를 연구하는 건 — 공허가 나를 연구하게 허락하는 거야”
**한 줄 훅**: 칸조차 내려가지 않는 최하층에서 어둠과 대화하는 존재 — 목소리만 있고 얼굴은 없다

### 🗝️ V-E “그림자의 기억” (히든)
**해금 조건**:  
1. 선봉(생츄어리) 호감 MAX  
2. 그림자 호감 50 % 이상 (독사 힌트로 유도)  
3. 최하층 2분간 대기

### 🌌 물리적·시스템적 월드 변화
| 선택지 | 그림자 반응 | 변화 | 연동 |
|---|---|---|---|
| A “후회하고 있어요” | 눈물 + 랜턴 1개 ON | 최하층에 희미한 빛 → 공허 파편 채굴 효율 +15 % | Echo Shard 1개 추가 지급 |
| B “정당했다고 생각해요” | 어둠 유지 | 대신 어둠 속 NPC 응답률 ↑ | 선봉과의 서신 교류 해금, 선봉 NPC 위치 실시간 공유 |

### 💰 그림자의 상점
- **물품**: 공허 연구 노트(100금화, 3회 한정) → Echo Shard 2개 소모 시 추가 페이지 해금
