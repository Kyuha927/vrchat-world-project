# 개선 지침

# 🛠️ 빌런 vs 히어로 9.5/10 달성을 위한 상세 개선 지침
> (K2 피드백 기반, 실현 가능 100% 수준)

## 1. 시간선형성 문제 해결 — 시즌 리셋 메타 시스템
### 🕰️ **「기억의 잔영(Memory Echo)」 시스템**
- **핵심 규칙**  
  시즌 종료 시 NPC 기억은 초기화되지만, 플레이어가 남긴 행적(선택, 킬, 퀘스트 완료)은 `Echo Shard`라는 아이템으로 남는다.  
  - Echo Shard는 다음 시즌에 해당 NPC에게 **대사 1~2개 + 퀘스트 보상 5% 상향**으로 반영.  
  - 한 NPC당 최대 3개까지 누적 → 3회째부터는 NPC가 “당신이 반복되는 악몽 같다”라고 인식, **숨검 퀘스트(히든 스킨)** 해금.  

- **데이터 구조**  
  ```json
  {
    "npcId": "vortex_khan",
    "echoes": [
      { "season": 3, "choice": "sacrificed_sister", "weight": 0.15 },
      { "season": 4, "choice": "spared_sister", "weight": 0.05 }
    ]
  }
  ```
- **UI 구현**  
  `/echo` 명령어 입력 시 개인 Echo 목록 GUI 출력 → 클릭하면 해당 시즌 클립 5초 자동 재생(ReplayMod 연동).

## 2. 빌런 vs 히어로 밸런스 재조정
### ⚔️ 전쟁터(Warfront) 이원화
- **히어로 전쟁터**: 석양 지역 북쪽 `정화의 심장(Heart of Purification)`  
  - 빌런 이동거리: 420블록 (지하철 1회 환승 필수)  
  - 히어로 이동거리: 180블록 (길이 직선)  
- **빌런 전쟁터**: 볼텍스 지역 남동 `타락의 심장(Heart of Corruption)`  
  - 히어로 이동거리: 440블록  
  - 빌런 이동거리: 160블록  

### 🌀 정화 스킬 **단일화**
- **기존**: 정화 = 공허 파편 획득 + 몬스터 스폰 억제(이중 이득)  
- **변경 후**:  
  - 정화 사용 시 **개인** 공허 파편 획득량 30% 상승  
  - **월드** 공허 파편 스폰율은 15% 감소 → 단체 플레이 시 전체 손해 > 개인 이득, 협동 유도.  
  - 쿨타임 6분 → 8분 (사용 빈도 ↓)

## 4. 신규 플레이어 10분 온보딩 퀘스트 3개
### 🔰 **「첫걸음」 튜토리얼 시리즈 (히어로 기준)**  
1. **선택의 방** (2분)  
   - NPC “가이드 엘라” 대화 → 4가지 질문(최애 무기, 선호 거리, 협동 vs 독공, 희생 vs 보존).  
   - 결과에 따라 초기 장비(검/활/지팡이) 지급 + **Echo Shard 1개** (시즌 0 흔적).  
2. **정화 시범** (3분)  
   - 근처 공허 블록 3개 정화 → 실시간 채팅 “/정화” 명령어 학습 → 초기 20개 파편 지급.  
3. **첫 전쟁터 참여** (5분)  
   - 자동 매칭 4인 파티 → AI 보스(더미) 처치 → 최종 **진영 배지(띠)** 장착.  
   - 퀘스트 완료 시 `/warp warfront` 명령어 잠금 해제.

> 빌런은 “타락 시범”으로 동일 구조, 대신 블록 **오염** 진행.

## 5. 금화 경제 시스템 설계
### 💰 **듀얼 화폐 구조**
| 화폐 | 획득처 | 주요 용도 | 인플레이션 방지 |
|---|---|---|---|
| **공허 파편** | 몬스터 처치, 월드 이벤트 | 고급 장비 제작, 진영 기여도 | 매주 획득량 20% 감소 디버프 |
| **금화** | 일일 퀘스트, 상점 판매, 스트리머 후원 | 수리, 소비품, 빠른 이동(워프 티켓) | 상점 판매 시 5% 수수료 + 금화→파편 환전 비율 주간 변동(0.9~1.2) |

- **상점 UI**  
  1 TAB: 금화 아이템 | 2 TAB: 파편 아이템 | 3 TAB: 환전소  
  환전소는 실시간 **유동 환율** 표시 → 스트리머 수익화(후원 시 금화 지급) 가능.

## 6. 로그아웃 페널티 시스템
### 😴 **「잊혀진 기억」 디버프**
- **기준**: 24시간 이상 미접속 시 발동.  
- **효과**:  
  - 진영 기여도 1%/일 감소 (최대 20%)  
  - 복귀 1시간 내 **기억 회복 퀘스트** 수행 시 전액 복구 + 추가 버프 10분.  
- **면제권**:  
  - 시즌 패스 소지자 7일, 스트리머 방송 중 연결 확인 시 즉시 면제.  
  - `/absence <사유>` 입력 시 48시간 면제 (월 2회).

## 7. 크로스 진영 협력 던전 2개
### 🌌 **던전 A: 「뒤틀린 심장」**
- **개요**: 빌런/히어로 6인(각 3) 파티, 15분 타이머  
- **핵심**:  
  - 1페이즈: 각 진영이 자신의 색상(빨강/파랑) **심장** 보호.  
  - 2페이즈: 중앙에 **뒤틀린 심장** 등장, 양 진영이 **동시** 체력 깎기 → 협력 실패 시 월드 전역 디버프 2시간.  
- **보상**:  
  - 성공 시 **협력 코인** 50개 (빌런+히어로 상점 모두 사용 가능)  
  - 실패 시 다음 주 재도전 가능, 실패 횟수 3회 누적 시 **히든 스킨** 해금 퀘스트 발생.

### 🏰 **던전 B: 「시간의 뒤틀림」**
- **개요**: 8인(4 vs 4) PvPvE, 3라운드  
- **룰**:  
  - 라운드마다 진영이 번갈아 **보스**와 **방어자** 역할.  
  - 보스 역할 팀은 **강화된 AI**로 조종, 방어자 팀은 실제 플레이어.  
  - 3라운드 총합 점수(처치 시간 + 생존자 수)로 승리 판정.  
- **보상**:  
  - 승리 진영: 전용 탈것(공중 부양 보드) 7일  
  - **MVP** 1인: 실시간 스트리머 채팅에 노출, 후원 수익 10% 추가.
