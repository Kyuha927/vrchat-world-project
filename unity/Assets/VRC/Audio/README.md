# 벨튀 대작전 — 사운드 에셋 목록

## 필요한 사운드 파일

아래 파일들을 이 폴더에 배치하세요. WAV 또는 OGG 형식 권장.
VRChat Quest 호환을 위해 파일 크기를 최소화하세요.

### SFX/ (효과음)
| 파일명 | 설명 | 추천 길이 |
|--------|------|-----------|
| `doorbell_ding.wav` | 초인종 벨 소리 | 0.5~1초 |
| `door_creak.wav` | 문 열리는 소리 | 1~2초 |
| `alarm_buzz.wav` | 잡혔을 때 경보음 | 1~2초 |
| `coin_ding.wav` | 점수 획득 효과음 | 0.3~0.5초 |
| `hide_enter.wav` | 숨을 때 효과음 | 0.5~1초 |
| `neighbor_angry.wav` | 주민 분노 반응 | 1~2초 |
| `elevator_move.wav` | 엘리베이터 이동 | 1.5~2초 |
| `elevator_ding.wav` | 엘리베이터 도착 | 0.5초 |
| `game_start.wav` | 게임 시작 사운드 | 2~3초 |
| `game_over.wav` | 게임 종료 사운드 | 2~3초 |

### BGM/ (배경음악)
| 파일명 | 설명 | 추천 길이 |
|--------|------|-----------|
| `bgm_cyberpunk.ogg` | 게임 중 BGM (루프) | 60~120초 |
| `bgm_lobby.ogg` | 로비 대기 BGM (루프) | 30~60초 |
